# Wolfman Macos1015-fixes

## How do I install these formulae?

`brew install wolfman/macos1015-fixes/<formula>`

Or `brew tap wolfman/macos1015-fixes` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "wolfman/macos1015-fixes"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
